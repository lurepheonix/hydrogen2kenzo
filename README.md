# Hydrogen to Kenzo rom porting tool

### This tool it designed to port xiaomi.eu roms from hydrogen (Mi Max) to kenzo/kate (Redmi Note 3 Snapdragon).

### How to use the script?
`./patch.sh mimaxrom.zip`

### Requirements
* Linux
* zip/unzip
* gzip/gunzip
* cpio
* openjdk8
* sudo
